### 内存结构
- SGA
    - shared pool
    - database buffer cache
    - redo log buffer
- PGA
- UGA
    - 共享服务器连接,UGA包含在SAG中
    - 专有服务器连接，UGA就包含在专有服务器的PGA中

- 用户进程
- 服务器进程
- 进程监控`PMON`
- 系统监控`SMON`
- 数据写入`DBWR`
- 日志写入`LGWR`
- 检查点进程`CKPT`
- 归档进程`ARCH`
